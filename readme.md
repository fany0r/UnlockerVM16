
## VMware 解锁安装 macOS

### 1、git clone

```bash
git clone https://github.com/Kuibagit/UnlockerVM16.git
cd UnlockerVM16
mkdir tools
```
### 2、下载 vmware tools包，保存到tools目录下。

下载地址：

a、官网
https://softwareupdate.vmware.com/cds/vmw-desktop/fusion/12.2.3/19436697/x86/core/com.vmware.fusion.zip.tar

b、蓝奏云：https://wwp.lanzouy.com/b02df196h 密码:1snz

分卷为7个压缩包，全部下载，删除后缀【.gz】，解压得到 [com.vmware.fusion.zip.tar] 放到tools目录下。

### 3、以管理员权限运行 `win-install.cmd` 

![](https://s1.ax1x.com/2022/07/03/j8RxuF.png)

![](https://s1.ax1x.com/2022/07/03/j8RjjU.png)

如需恢复破解，运行 `win-uninstall.cmd`
